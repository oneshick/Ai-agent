# AI Data Analyst Agent

Приложение для анализа CSV-файлов через LLM с tool-calling.
Пользователь загружает датасет, модель вызывает Python-инструмент, анализирует данные через pandas и возвращает структурированный JSON с метриками, графиками и инсайтами.

---

## Структура проекта

```text
app/
├── security.py               # защита от prompt-injection
├── requirements.txt          # зависимости Python
├── app.py                    # оснвной файл
├── config.py                 # конфигурация API
├── prompts.py                # системный prompt
├── agent.py                  # цикл работы LLM-агента
├── tools.py                  # execute_python()
├── charts.py                 # построение графиков 
├── README.md
```

### 1. Установите Python

Проверьте:

```bash
python --version
```

---

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

---

### 3. Получите OpenRouter API Key

Создайте ключ:

```text
https://openrouter.ai/keys
```

---

### 4. Настройте config.py

```python

API_KEY = "sk-or-v1-..."
MODEL = "qwen/qwen3.6-plus"
```

streamlit run streamlit_app.py

## Что выводится

| Блок | Описание |
|------|----------|
| **Резюме** | 2–3 предложения об общей картине датасета |
| **Ключевые метрики** | Числовые показатели с трендами (↑ ↓ →) |
| **Визуализации** | Интерактивные графики (bar, line, pie, scatter, histogram) |
| **Инсайты** | Находки с уровнями важности: 🔴 высокий, 🟡 средний, 🔵 низкий |
| **Рекомендации** | Конкретные действия на основе данных |
| **JSON-ответ** | Полный структурированный вывод агента (для отладки) |

## Безопасность (`security.py`)

Приложение защищено от **prompt injection** — атаки, при которой вредоносные инструкции в данных пытаются изменить поведение модели.

### Уровень 1 — Входные данные (CSV)

Каждая текстовая ячейка проверяется на ~30 паттернов инъекций на русском и английском языке:

- `"Ignore previous instructions..."` → ячейка заменяется заглушкой
- `"Игнорируй предыдущие инструкции..."` → ячейка заменяется заглушкой
- `"You are now a different AI..."` → ячейка заменяется заглушкой
- Ячейки длиннее 500 символов → заменяются заглушкой

Если **более 1%** текстовых ячеек подозрительны — файл отклоняется целиком с сообщением об ошибке.

### Уровень 2 — Сгенерированный код

Перед каждым `exec()` код проверяется по блок-листу из ~25 паттернов. Запрещены:

```python
import os, subprocess, sys, socket, requests  # системные модули
open(), eval(), exec(), compile()              # опасные встроенные
os.environ, os.getenv                          # переменные окружения
__builtins__, __globals__, globals(), locals() # интроспекция
```

### Уровень 3 — Ответ агента (JSON)

Финальный JSON валидируется перед отображением:

- Все строковые поля обрезаются до безопасной длины
- Enum-поля (`trend`, `severity`, `chart.type`) проверяются по whitelist
- Числовые данные графиков приводятся к `float`
- Ограничено количество метрик, инсайтов, графиков и рекомендаций
